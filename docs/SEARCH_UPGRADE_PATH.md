# مسار ترقية البحث

الوضع الحالي: بحث Prisma `contains` مع فلاتر server-side وروابط قابلة للمشاركة.

## المرحلة 1

- تحسين الفهارس الحالية على `status`, `city`, `jobType`, `jobCategory`.
- إبقاء كل المنطق داخل `src/lib/search/jobs.ts`.

## المرحلة 2

- إضافة PostgreSQL Full Text Search على العنوان والوصف والمتطلبات والمهارات.
- دعم قاموس عربي بسيط عبر normalization يدوي للكلمات الشائعة.

## المرحلة 3

- تشغيل Meilisearch أو Typesense عند نمو عدد الوظائف.
- فهرسة `Job` عند النشر والتعديل.
- ترتيب النتائج حسب featured/urgent ثم relevance.

## لماذا ليس الآن؟

الـ MVP يجب أن يعمل محليًا بلا خدمات مدفوعة أو سيرفرات إضافية.
