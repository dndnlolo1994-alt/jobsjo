# تشغيل لوحة الأدمن

الأدمن يستطيع:

- إنشاء وظيفة يدوية.
- مراجعة جودة الوظيفة.
- رؤية تنبيهات تكرار.
- إدارة المصادر.
- تعليم المدفوعات paid/waived/unpaid.
- مراجعة مطالبات الشركات.
- مشاهدة المستخدمين والطلبات والبلاغات.

قبل نشر أي وظيفة منسّقة:

- تحقق من المصدر.
- ضع اسم مصدر واضح.
- ضع تاريخ انتهاء.
- لا تنسخ بيانات خاصة.
- اترك رابط المصدر إن كان عامًا.
