# نظام CV وPDF

المسارات:

- `/me/cv`: بناء وحفظ السيرة.
- `/me/cv/preview`: معاينة A4.
- `/me/cv/download`: صفحة طباعة PDF.

القرار الحالي:

استخدام HTML/CSS print لأنّه موثوق محليًا وخفيف على Vercel. المستخدم يفتح صفحة التنزيل ويستخدم أمر الطباعة في المتصفح ثم "حفظ كملف PDF".

بوابة الدفع:

- السجل اليدوي من نوع `CV_PDF`.
- السعر 2 دينار.
- المتغير `REQUIRE_CV_PAYMENT` يتحكم في gating.
- الأدمن يستطيع تعليم السجل `PAID` أو `WAIVED`.

مسار الترقية:

إضافة server PDF عبر Playwright/Puppeteer عندما تتوفر بنية مناسبة لتشغيل Chromium.
